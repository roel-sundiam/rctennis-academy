# 🎾 Angular App Enhancement -- Tennis Academy Website + Booking System (Mobile-Friendly)

## 📌 Goal

Enhance an existing Angular booking system into a **full mobile-friendly
website** with a landing page while keeping the booking system intact.

Also, incorporate the **academy logo** into the design (navbar and hero
section).

------------------------------------------------------------------------

## 🧠 Requirements

### 1. Landing Page (Main Page `/`)

Create a **modern, mobile-responsive landing page**.

#### 🏠 Hero Section

-   Display academy logo prominently
-   Title: "Renell Crescini Tennis Academy"
-   Subtitle: "Train. Play. Compete."
-   CTA Button: "Book a Session" → `/booking`

------------------------------------------------------------------------

#### 🎾 Services Section

Display:

-   10 & Under Tennis
-   Adult Tennis Development
-   Complete Junior Tennis Development
-   Over 40's Strength & Conditioning
-   High Performance Player Development
-   1-on-1 or Group Lessons
-   Tennis Clinics & Events
-   Coaching Certification Programs

------------------------------------------------------------------------

#### 👤 About Section

"Professional tennis coaching for all levels, focused on development,
performance, and community."

------------------------------------------------------------------------

#### 📞 Contact Section

-   Location: Angeles City
-   Contact number
-   Social links

------------------------------------------------------------------------

### 2. Booking System Integration

-   Keep existing booking system
-   Route: `/booking`
-   Navigation from landing page

------------------------------------------------------------------------

### 3. Navigation Bar

-   Include logo on the left
-   Home → `/`
-   Services → scroll
-   Booking → `/booking`
-   Contact → scroll

------------------------------------------------------------------------

### 4. Routing

-   `/` → Landing Page
-   `/booking` → Booking system
-   `/admin` → Admin panel

------------------------------------------------------------------------

### 5. Mobile-Friendly UI/UX (IMPORTANT)

-   Fully responsive design
-   Use Flexbox or Grid
-   Large touch-friendly buttons
-   Sticky navbar
-   Proper spacing and readable fonts
-   Optimize for mobile-first design

------------------------------------------------------------------------

### 6. Deliverables

-   landing-page.component.ts/html/css
-   Updated routing module
-   Navbar with routerLink
-   Responsive layout
-   Logo integration (image asset handling)

------------------------------------------------------------------------

## 🎯 Important

-   Do NOT break booking system
-   Focus on UI + routing only

------------------------------------------------------------------------

## 🚀 Result

-   Landing page as main website
-   Booking system accessible via button
-   Mobile-friendly design
-   Logo fully integrated into UI
